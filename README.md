# Bluecurve-Qt
Red Hat Bluecurve theme ported to Qt 5/6.

## Requirements
- Qt 5/6 qtbase development files.
- KDE Extra CMake Modules package.

## Installation

#### 1. Clone the repository
```
git clone https://github.com/neeeeow/Bluecurve-Qt.git
cd Bluecurve-Qt
```

#### 2. Compile & install
```
mkdir build && cd build
cmake ..
make
sudo make install
```
#### (Optional) 3. Install the GTK 3/4 and icon theme
See [neeeeow/Bluecurve](https://github.com/neeeeow/Bluecurve).
